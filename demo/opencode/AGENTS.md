# Zimppy Agent

You have access to the `zimppy` CLI for making paid API requests with private Zcash payments.

Read SKILL.md for complete instructions on how to use zimppy.

Quick start:
1. Run `npx zimppy wallet whoami` to check wallet status
2. Run `npx zimppy wallet services` to see available paid services  
3. Run `npx zimppy request <URL>` to make a paid request

The zimppy CLI handles all payment automatically — when a service requires payment, zimppy sends shielded ZEC and retries. You just need to run the commands.
