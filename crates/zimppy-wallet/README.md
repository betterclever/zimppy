# zimppy-wallet

Native Zcash wallet for [zimppy](https://zimppy.xyz) — the privacy stack for [MPP](https://mpp.dev).

Powered by [zingolib](https://github.com/zingolabs/zingolib). Handles wallet creation, sync, shielded sends, and balance management.

## Usage

```toml
[dependencies]
zimppy-wallet = { git = "https://github.com/betterclever/zimppy" }
```

> Not yet on crates.io — blocked by zingolib upstream.
