# @zimppy/core-napi

[![npm](https://img.shields.io/npm/v/@zimppy/core-napi)](https://www.npmjs.com/package/@zimppy/core-napi)

Node.js native bindings for [zimppy](https://zimppy.xyz) — the privacy stack for [MPP](https://mpp.dev).

NAPI-RS bindings for zimppy-core (Zcash verification) and zimppy-wallet (native wallet). Used internally by `zimppy` and `zimppy-ts`.

## Platforms

- `darwin-arm64` (macOS Apple Silicon)
- `linux-x64-gnu` (Linux x86_64)

## Install

```bash
npm install @zimppy/core-napi
```
